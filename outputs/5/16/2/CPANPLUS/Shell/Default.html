    NAME

    CPANPLUS::Shell::Default - the default CPANPLUS shell

    SYNOPSIS

    1.     ### loading the shell:
    2.     $ cpanp                     # run 'cpanp' from the command line
    3.     $ perl -MCPANPLUS -eshell   # load the shell from the command line
    6.     use CPANPLUS::Shell qw[Default];        # load this shell via the API
    7.                                             # always done via CPANPLUS::Shell
    9.     my $ui = CPANPLUS::Shell->new;
    10.     $ui->shell;                             # run the shell
    11.     $ui->dispatch_on_input( input => 'x');  # update the source using the
    12.                                             # dispatch method
    14.     ### when in the shell:
    15.     ### Note that all commands can also take options.
    16.     ### Look at their underlying CPANPLUS::Backend methods to see
    17.     ### what options those are.
    18.     cpanp> h                 # show help messages
    19.     cpanp> ?                 # show help messages
    21.         cpanp> m Acme            # find acme modules, allows regexes
    22.         cpanp> a KANE            # find modules by kane, allows regexes
    23.         cpanp> f Acme::Foo       # get a list of all releases of Acme::Foo
    25.         cpanp> i Acme::Foo       # install Acme::Foo
    26.         cpanp> i Acme-Foo-1.3    # install version 1.3 of Acme::Foo
    27.         cpanp> i <URI>           # install from URI, like ftp://foo.com/X.tgz
    28.         cpanp> i <DIR>           # install from an absolute or relative directory
    29.         cpanp> i 1 3..5          # install search results 1, 3, 4 and 5
    30.         cpanp> i *               # install all search results
    31.         cpanp> a KANE; i *;      # find modules by kane, install all results
    32.         cpanp> t Acme::Foo       # test Acme::Foo, without installing it
    33.         cpanp> u Acme::Foo       # uninstall Acme::Foo
    34.         cpanp> d Acme::Foo       # download Acme::Foo
    35.         cpanp> z Acme::Foo       # download & extract Acme::Foo, then open a
    36.                                                           # shell in the extraction directory
    38.         cpanp> c Acme::Foo       # get a list of test results for Acme::Foo
    39.         cpanp> l Acme::Foo       # view details about the Acme::Foo package
    40.         cpanp> r Acme::Foo       # view Acme::Foo's README file
    41.         cpanp> o                 # get a list of all installed modules that
    42.                                                           # are out of date
    43.         cpanp> o 1..3            # list uptodateness from a previous search
    45.         cpanp> s conf            # show config settings
    46.         cpanp> s conf md5 1      # enable md5 checks
    47.         cpanp> s program         # show program settings
    48.         cpanp> s edit            # edit config file
    49.         cpanp> s reconfigure     # go through initial configuration again
    50.         cpanp> s selfupdate      # update your CPANPLUS install
    51.         cpanp> s save            # save config to disk
    52.         cpanp> s mirrors         # show currently selected mirrors
    54.         cpanp> ! [PERL CODE]     # execute the following perl code
    56.         cpanp> b                 # create an autobundle for this computers
    57.                                                           # perl installation
    58.         cpanp> x                 # reload index files (purges cache)
    59.         cpanp> x --update_source # reload index files, get fresh source files
    60.         cpanp> p [FILE]          # print error stack (to a file)
    61.         cpanp> v                 # show the banner
    62.         cpanp> w                 # show last search results again
    64.         cpanp> q                 # quit the shell
    65.         cpanp> e                 # exit the shell and reload
    67.         cpanp> /plugins          # list available plugins
    68.         cpanp> /? PLUGIN         # list help test of <PLUGIN>
    70.     ### common options:
    71.     cpanp> i ... --skiptest # skip tests
    72.     cpanp> i ... --force    # force all operations
    73.     cpanp> i ... --verbose  # run in verbose mode

    DESCRIPTION

    This module provides the default user interface to CPANPLUS . You can start it via the cpanp binary, or as detailed in the SYNOPSIS.
