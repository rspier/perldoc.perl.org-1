    NAME

    TAP::Parser::SourceHandler::Executable - Stream output from an executable TAP source

    VERSION

    Version 3.38

    SYNOPSIS

    1.   use TAP::Parser::Source;
    2.   use TAP::Parser::SourceHandler::Executable;
    4.   my $source = TAP::Parser::Source->new->raw(['/usr/bin/ruby', 'mytest.rb']);
    5.   $source->assemble_meta;
    7.   my $class = 'TAP::Parser::SourceHandler::Executable';
    8.   my $vote  = $class->can_handle( $source );
    9.   my $iter  = $class->make_iterator( $source );

    DESCRIPTION

    This is an executable TAP::Parser::SourceHandler - it has 2 jobs:

    1. Figure out if the TAP::Parser::Source it's given is an executable command (can_handle).

    2. Creates an iterator for executable commands (make_iterator).

    Unless you're writing a plugin or subclassing TAP::Parser, you probably won't need to use this module directly.

    METHODS

    Class Methods

    can_handle 

    1.   my $vote = $class->can_handle( $source );

    Only votes if $source looks like an executable file. Casts the following votes:

    1.   0.9  if it's a hash with an 'exec' key
    2.   0.8  if it's a .bat file
    3.   0.75 if it's got an execute bit set

    make_iterator 

    1.   my $iterator = $class->make_iterator( $source );

    Returns a new TAP::Parser::Iterator::Process for the source. $source->raw must be in one of the following forms:

    1.   { exec => [ @exec ] }
    3.   [ @exec ]
    5.   $file

    croak s on error.

    iterator_class

    The class of iterator to use, override if you're sub-classing. Defaults to TAP::Parser::Iterator::Process.

    SUBCLASSING

    Please see SUBCLASSING in TAP::Parser for a subclassing overview.

    Example

    1.   package MyRubySourceHandler;
    3.   use strict;
    5.   use Carp qw( croak );
    6.   use TAP::Parser::SourceHandler::Executable;
    8.   use base 'TAP::Parser::SourceHandler::Executable';
    10.   # expect $handler->(['mytest.rb', 'cmdline', 'args']);
    11.   sub make_iterator {
    12.     my ($self, $source) = @_;
    13.     my @test_args = @{ $source->test_args };
    14.     my $rb_file   = $test_args[0];
    15.     croak("error: Ruby file '$rb_file' not found!") unless (-f $rb_file);
    16.     return $self->SUPER::raw_source(['/usr/bin/ruby', @test_args]);
    17.   }
