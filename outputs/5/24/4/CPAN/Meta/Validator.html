    NAME

    CPAN::Meta::Validator - validate CPAN distribution metadata structures

    VERSION

    version 2.150005

    SYNOPSIS

    1.   my $struct = decode_json_file('META.json');
    3.   my $cmv = CPAN::Meta::Validator->new( $struct );
    5.   unless ( $cmv->is_valid ) {
    6.     my $msg = "Invalid META structure.  Errors found:\n";
    7.     $msg .= join( "\n", $cmv->errors );
    8.     die $msg;
    9.   }

    DESCRIPTION

    This module validates a CPAN Meta structure against the version of the the specification claimed in the meta-spec field of the structure.

    METHODS

    new

    1.   my $cmv = CPAN::Meta::Validator->new( $struct )

    The constructor must be passed a metadata structure.

    is_valid

    1.   if ( $cmv->is_valid ) {
    2.     ...
    3.   }

    Returns a boolean value indicating whether the metadata provided is valid.

    errors

    1.   warn( join "\n", $cmv->errors );

    Returns a list of errors seen during validation.
